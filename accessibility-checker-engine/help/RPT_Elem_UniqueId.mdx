### Why is this important?

The `id` attribute provides a machine-readable identifier for the elements of the page. A unique value for `id` allows browsers and assistive technologies to interpret and parse the content accurately.

<div id="locSnippet"></div>

### What to do

 * For each element with an `id` attribute, provide a unique, non-empty value.

For example:

<CodeSnippet type="multi" light={true}>&lt;h1 id="news"&gt;Breaking news&lt;/h1&gt;
&lt;p id="newsintro"&gt;The latest stories from our correspondents around the world.&lt;/p&gt;
...
&lt;div role="article" aria-labelledby="news" aria-describedby="newsintro"&gt;
    ...
&lt;/div&gt;</CodeSnippet>
