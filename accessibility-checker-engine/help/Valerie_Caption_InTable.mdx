## Who does this affect?

 * Blind people using screen readers

## Why is this important?

The caption for a table is an identifier and acts like a title or heading for the table. The `<caption>` element allows screen readers to navigate directly to the caption to get the description of the table.

</Column>

## `<caption>` element is not nested inside a `<table>` element
`<caption>` element must be nested inside the associated `<table>` element

[IBM 1.3.1 Info and Relationships](https://www.ibm.com/able/guidelines/ci162/info_and_relationships.html) | [WCAG 2.1 technique H39](https://www.w3.org/WAI/WCAG21/Techniques/html/H39)


<div id="locSnippet"></div>

## What to do

 * Nest the `<caption>` element inside the associated `<table>` element. For example:

```
<table id="data_table1" border="1" summary="A simple data table showing the number of boys and girls in each grade of an elementary school">
        <caption>Boys and Girls in Elementary School Classes</caption>
        <tr> ... </tr>
        <tr> ... </tr>
</table>
```
