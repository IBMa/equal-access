### Label text is located after its associated text input or `<select>` element

<div id="locLevel"></div>

Text input and `<select>` elements must have the label before the element

</Column>
</Row>
<Row>
<Column colLg={11} colMd={5} colSm={4} className="toolMain">

### Why is this important?

Applications generally place the labels for text input and `<select>` elements before the element itself. Users expect this convention to be followed consistently. When labels are placed after the text input or `<select>` element, users can become disoriented.

<div id="locSnippet"></div>

### What to do

* Place the text of the label immediately before the text input or `<select>` element.

For example:

<CodeSnippet type="single" light={true}>&lt;label for="txt1"&gt;Field 1&lt;/label&gt; &lt;input type="text" id="txt1" name="txt1"&gt;</CodeSnippet>

</Column>

### Who does this affect?

* Blind people using screen readers
* People with low vision using screen magnification
